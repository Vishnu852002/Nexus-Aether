# Nexus-Aether

Nexus-Aether is a local-first AI desktop app with a privacy-focused runtime and multi-platform installers.

## What To Download

Go to **Releases** and pick exactly one file for your OS:

- **Windows:** `Nexus-Aether.Setup.<version>.exe`
- **macOS (Apple Silicon):** `Nexus-Aether-<version>-arm64.dmg`
- **Linux (Debian/Ubuntu):** `nexus_aether_<version>_amd64.deb`
- **Linux (portable):** `Nexus-Aether-<version>.AppImage`

## For Developers

- `nexus-aether-core-<tag>.zip` is the OSS runtime/core package.

## Ignore These Assets

- `latest*.yml` and `*.blockmap` are auto-update metadata.
- `Source code (zip/tar.gz)` are GitHub auto-generated source archives.
